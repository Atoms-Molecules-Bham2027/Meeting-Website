# Conference Website for GitHub Pages

This is a static conference website designed to deploy directly on GitHub Pages. It includes:

- Home
- Programme
- Registration
- Accommodation
- Venue and Travel Information
- Code of Conduct

## Quick Setup

1. Copy this folder's contents into a GitHub repository.
2. Edit the conference name, dates, venue, fees, and contact details in the `.html` files.
3. In GitHub, go to `Settings -> Pages`.
4. Choose `Deploy from a branch`.
5. Select the branch containing these files and the root folder.
6. Save. GitHub Pages will publish the site.
